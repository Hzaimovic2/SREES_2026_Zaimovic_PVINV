# Dokumentacija

Ovdje ide LyX izvjestaj za seminarski rad.

Predlozena struktura izvjestaja:

1. Uvod i cilj rada
2. Model solarne elektrane
3. Model invertera i upravljanje aktivnom/reaktivnom snagom
4. XML konfiguracija modela
5. Testni slucajevi: case9, case30, case118, case300
6. C++ implementacija, CMake i thread-ovi
7. natID/dTwin plugin integracija
8. Rezultati, grafici i diskusija
9. Zakljucak
10. Literatura

Pomocni dokumenti:

- `setup_status.md`: sta je instalirano i kako je buildano
- `test_results.md`: rezultati za case9/case30/case118/case300
- `manual_dtwin_check.md`: rucna provjera plugin-a u dTwin-u
- `what_to_explain.md`: kratka priprema za odbranu
